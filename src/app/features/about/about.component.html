À propos

Application personnelle de planification de repas, entièrement locale.

Objectif

Ce planificateur vous aide à organiser vos déjeuners et dîners sur une semaine, à créer ou importer des recettes, et à générer une liste de courses agrégée selon le nombre de portions prévues.

Fonctionnalités

  • Grille hebdomadaire (7 jours × 2 repas : midi et soir)
  • Création de recettes avec ingrédients quantifiés
  • Import de recettes via l'API publique TheMealDB
  • Liste de courses dynamique avec cases à cocher
  • Ajustement des portions par créneau
  • Persistance locale dans le navigateur (localStorage)

Confidentialité

Aucune donnée n'est envoyée vers un serveur tiers : pas de compte, pas de cloud, pas de télémétrie. Seules les requêtes vers TheMealDB sont effectuées lors d'un import de recette. Vos recettes et votre planning restent sur votre appareil.

Technologies

Application construite avec Angular (composants standalone), signals pour l'état réactif, Reactive Forms et SCSS. Les données sont stockées localement via localStorage.
